# **개인정보처리방침 (Privacy Policy)**

* 이 GPT는 사용자의 요청에 따라 GitHub 저장소의 공개 JSON 데이터를 조회할 수 있습니다.
* 이 GPT는 사용자의 개인 식별 정보(이름, 이메일, 전화번호 등)를 수집하거나 저장하지 않습니다.
* 사용자가 입력한 대화 내용은 답변 생성 및 기능 수행(예: 공개 JSON 조회)에만 사용됩니다.
* 외부로 전송되는 데이터는 Actions 호출에 필요한 요청 정보(예: 조회할 파일명/키)로 제한됩니다.
* 이 GPT는 결제 정보, 의료/민감정보를 수집하지 않습니다.
* 문의: GitHub Issues
